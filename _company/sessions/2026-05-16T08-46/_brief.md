# 📋 작업 브리프

**원 명령:** Restaurant owners struggle with menu updates across platforms 에 대한 문제를 해결할수 있는 
웹 사이트 기획해서 웹사이트 만들어줘

## 요약
식당 메뉴 업데이트 문제 해결을 위한 웹사이트 기획 및 제작 - 경쟁사 분석, 디자인 브리프, 웹 개발 계획

## 분배
- **🔍 Researcher**: 현재 레스토랑 메뉴 관리 SaaS 시장 주요 경쟁사 (MenuCarta, TouchBistro 등) 기능 비교 및 API 통합 가능성 조사. Google Maps, Yelp, OpenTable等平台의 메뉴 업데이트 정책 연구.
- **🎨 Designer**: 식당용 메뉴 관리 툴 디자인 브리프 작성 - 브랜드 컬러(신뢰감/맛깔스러운 오렌지-초록 계열), 타이포그래피(가독성 높은 산세리프), 썸네일 컨셉 (메뉴판 미리보기)
- **✍️ Writer**: 웹사이트 카피라이팅: 핵심 가치 제안(한 곳에서 모든 플랫폼 메뉴 동기화), 후크 문구('하루 10분으로 메뉴 업데이트 끝'), 블로그 포스트 개요 작성
- **💻 코다리**: MERN 스택(React + Node.js) 또는 Next.js 기반으로 메뉴 관리 웹앱 개발 계획 - API 구조 설계, 데이터베이스 스키마, 프론트엔드 컴포넌트 아키텍처
- **💼 현빈**: 수익화 전략: 구독 모델(무료/프로/엔터프라이즈), 파트너십 (플랫폼 연동 수수료), KPI 설정(MAU, 전환율, 평균 계약 기간)
